Description détaillée

Structure representant une main de cartes.

Paramètres:
Le champ tabMain est un tableau de pointeurs sur Carte.
Le champ nCarte est un entier representant le nombre de pointeurs sur Carte valides dans le tableau, c'est a dire le nombre de cartes contenues dans la main.
Note:
La main peut contenir de 0 à 5 cartes.
